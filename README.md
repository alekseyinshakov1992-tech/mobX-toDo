# Приложение toDo List На TS + React + MobX

Техническое задание:

Приложение имеет несколько страниц: главная и выполненное. Навигация находится в шапке профиля.

Главная страница содержит кнопку "Создать дело" и список созданных дел.

При нажатии на кнопку создать, появляется форма создания

Форма содержит поле заголовка, поле описания и кнопки "Создать" и "Отмена"

Каждый элемент списка на главной странице содержит следующие элементы: заголовок, описание, кнопку выполнено и удалить

При нажатии на кнопку удалить появляется модальное окно с подтверждением удаления.

На странице выполнено элементы списка имеют кнопки вернуть и удалить

Хранение данных и работа с ними происходит в mobX

Язык и тема меняется с помощью контекста
